# Setup OpenClaw

Dokumen ini menjelaskan cara menghubungkan project chatbot dengan OpenClaw tanpa memasukkan konfigurasi pribadi ke GitHub.

## Lokasi Lokal

Contoh lokasi OpenClaw di WSL:

```txt
/home/afiyantorfid/.openclaw
/home/afiyantorfid/.nvm/versions/node/v22.16.0/lib/node_modules/openclaw
```

Folder `.openclaw` kemungkinan berisi konfigurasi, token, session, atau data runtime. Jangan commit folder tersebut.

## Yang Aman Dicantumkan

- Cara install OpenClaw.
- Versi OpenClaw yang digunakan.
- Contoh konfigurasi tanpa secret.
- Penjelasan peran OpenClaw dalam arsitektur.
- Screenshot hasil integrasi.

## Yang Tidak Aman Dicantumkan

- Token Telegram asli.
- API key OpenRouter.
- File `openclaw.json` asli.
- Backup konfigurasi OpenClaw.
- Session, cache, atau log percakapan yang mengandung data pribadi.
